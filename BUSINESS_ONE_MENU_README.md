# Business One Service Menu App

A modern, responsive digital menu application for Business One's integrated business solutions. This web-based application provides an interactive interface for showcasing Business One's services including POS systems, Payment Processing, Phone Service, and Website Development.

## Features

- **Interactive Service Cards**: Browse through Business One's services with detailed information
- **Service Details Modal**: Click on any service to view comprehensive details and features
- **Settings Panel**: Customize your experience with theme options and display preferences
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Dark Mode Support**: Choose between light, dark, or auto theme based on system preferences
- **Local Storage**: Your preferences are saved and persist across sessions

## Files

- `business-one-menu.html` - Main HTML structure
- `business-one-menu.css` - Styling and responsive design
- `business-one-menu.js` - Interactive functionality and service data

## Usage

1. Open `business-one-menu.html` in a web browser
2. Browse services by clicking on service cards
3. Access settings via the gear icon in the header
4. Customize your experience with theme and display options

## Services Included

1. **Point of Sale (POS)** - Modern POS systems for sales and inventory management
2. **Payment Processing** - Secure payment processing solutions
3. **Phone Service** - Business phone systems with advanced features
4. **Website Development** - Professional website design and development

## Customization

### Branding Colors
The app uses Business One's brand colors defined in CSS variables:
- Primary: `#1a4d7a` (Blue)
- Accent: `#ff6b35` (Orange)
- Customize these in `business-one-menu.css` under `:root`

### Adding Services
Edit the `services` array in `business-one-menu.js` to add or modify services.

### Contact Information
Update contact details in:
- Settings modal (HTML)
- JavaScript contact functions
- Footer section

## Browser Support

- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
- Mobile browsers (iOS Safari, Chrome Mobile)

## Notes

- This is a standalone web application
- No backend server required
- All data is stored locally in the browser
- Contact links open default email client
