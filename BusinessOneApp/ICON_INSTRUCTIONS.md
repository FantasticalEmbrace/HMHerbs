# App Icon Instructions

## Current Status
The app uses default Android launcher icons. To customize with Business One branding:

## Option 1: Use Android Studio Icon Generator

1. In Android Studio, right-click `res` folder
2. Select `New` → `Image Asset`
3. Choose "Launcher Icons (Adaptive and Legacy)"
4. Upload your Business One logo
5. Adjust settings and click "Next" → "Finish"

## Option 2: Manual Icon Creation

Create icons in these sizes:
- **mipmap-mdpi**: 48x48 px
- **mipmap-hdpi**: 72x72 px
- **mipmap-xhdpi**: 96x96 px
- **mipmap-xxhdpi**: 144x144 px
- **mipmap-xxxhdpi**: 192x192 px

Place files as:
- `ic_launcher.png` (square icon)
- `ic_launcher_round.png` (round icon)

## Quick Solution
For now, the app will use default Android icons. The app will function normally - you can add custom icons later.
