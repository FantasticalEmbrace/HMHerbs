Business One Support Agent

Install on each Windows register PC for remote support via RustDesk.

Setup (3 steps)

  1. In Admin → POS → Remote support, set POS_SUPPORT_ENROLL_KEY in server .env and restart the backend.
  2. Fill in the form below and copy the install command.
  3. On the register PC, open PowerShell as Administrator, paste the command, and press Enter.

Install command builder

Or download scripts and run install.ps1 manually — see README.

Run in PowerShell (Administrator)
